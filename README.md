# SOS Project

O Service Order System (SOS) é uma aplicação web destinada a auxiliar técnicos, sejam de 
grandes empresas ou autônomos, na organização, controle e acompanhamento de ordens de serviço 
de seus clientes. A aplicação permite o cadastro de clientes, serviços e ordens de serviço,
possibilitando o controle do status de cada atendimento. O sistema visa facilitar a gestão de
ordens de serviço de forma fácil, rápida e segura para diversos técnicos. Basta criar uma conta
para você, para sua equipe ou para sua empresa e começar a utilizar o sistema. Cada conta pode
ter seus próprios clientes, serviços e ordens de serviço, garantindo a privacidade e a organização
dos dados. Lembre-se, cada usuário tem acesso apenas aos dados da sua própria conta e aos cadastros
realizados por ele.
