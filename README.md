# SOS Project Front

Este é o projeto do front-end do Service Order System, desenvolvido utilizando o Angular Cli. O objetivo deste sistema é
oferecer aos usuários uma forma de gerenciar as ordens de serviço da sua empresa de forma fácil, fácil e segura.
